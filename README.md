# Spark com Scala - Projeto de Estudos

## 📌 Sobre o Projeto
Este repositório foi desenvolvido com o objetivo de estudar, praticar e aprimorar conhecimentos no ecossistema **Apache Spark utilizando a linguagem Scala**. Ao longo do projeto, foram exploradas a criação de funções, o uso de bibliotecas nativas, a manipulação eficiente de grandes volumes de dados e a aplicação das melhores práticas de mercado recomendadas para a linguagem.

## 🌿 Estrutura do Repositório (Branches)
Para manter o aprendizado organizado e modular, o código deste repositório não está centralizado na branch principal (`main`). Em vez disso, o projeto foi estruturado utilizando **branches específicas para cada assunto**, permitindo consultas isoladas e focadas em cada tópico abordado no curso.

A branch `main` contém a configuração do projeto base construída durante as **Aulas 1 a 6**. As demais aulas não listadas abaixo correspondem a conteúdos puramente teóricos.

Confira abaixo o mapeamento das aulas e suas respectivas branches:

| Aula | Tópicos Abordados | Branch Correspondente |
| :---: | :--- | :--- |
| **Aula 7** | Funções de Coluna (Col, Lit e Concat) | `feature/column-functions` |
| **Aula 8** | Expressões SQL (SQL Expressions) | `feature/sql-expressions` |
| **Aula 9** | Exercício Prático / Desafio (Assignment I) | `feature/assignment-i` |
| **Aula 11** | Agrupamentos e Ordenação (GroupBy, Sort & Aggregations) | `feature/groupby-sort-aggregations` |
| **Aula 12** | Funções de Janela (Window Functions) | `feature/window-functions` |

## 🛠️ Tecnologias Utilizadas
* **Scala**
* **Apache Spark** (Spark SQL / Dataset API)
* **SBT** (Simple Build Tool)
